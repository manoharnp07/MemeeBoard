WHERE TO GET THE MP3 FILES
===========================
Go to https://www.myinstants.com and search each sound name.
Click the download button (arrow icon) on the sound page.
Rename the file to match exactly what's listed above.
Put all files inside the "sounds" folder.

ALTERNATIVELY — when you open index.html in a browser and click
a missing sound, it will show a clickable download link at the
bottom of the page. Click it, save the file into the sounds/ folder.

DEPLOY TO NETLIFY
=================
1. Make sure all MP3s are in the sounds/ folder
2. Go to netlify.com/drop
3. Drag the entire "meme-board" FOLDER (not just index.html)
4. Done — live URL in seconds
